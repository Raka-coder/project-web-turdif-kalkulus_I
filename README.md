## Prerequisites
- node => v20.10.0
- npm  => v10.2.3

## Installation

Install my-project with npm

```bash
  npm install project-app
  cd project-app
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/Raka-coder/project-web-turdif-kalkulus_I.git
```

Go to the project directory

```bash
  cd project-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
